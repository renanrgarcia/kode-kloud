## Create Virtual Machine

- Open Oracle Virtual Box
- Click on New
- Name: my-centos
- Type: Linux
- Version: Red Hat (64-bit)
- Memory: 1 GB
- Use an existing virtual hard disk file, using the CentOS image.
- Click on Start
